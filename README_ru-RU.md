# TypeK - Инструмент голосового ввода

[Читать на других языках: [繁體中文](README.md) | [English](README_en-US.md) | [日本語](README_ja-JP.md) | [简体中文](README_zh-CN.md) | [한국어](README_ko-KR.md) | [Français](README_fr-FR.md) | [Español](README_es-ES.md) | [Русский](README_ru-RU.md) | [ไทย](README_th-TH.md) | [Tiếng Việt](README_vi-VN.md) | [العربية](README_ar-SA.md)]

> Удерживайте, чтобы говорить, отпустите, чтобы испытать совершенно иное качество вывода — настольное приложение перевода голоса в текст.

TypeK — это кроссплатформенный настольный инструмент для голосового ввода. Зажмите горячую клавишу в любом приложении, чтобы начать говорить, отпустите ее — и голос будет транскрибирован через Groq Whisper API, а затем Groq LLM автоматически превратит вашу устную речь в связный письменный текст, вставляя его прямо на место курсора. Как будто у вас есть невидимый секретарь-редактор.

## Возможности

- **От устной речи к письменной** — ИИ автоматически удаляет слова-паразиты, реструктурирует предложения, исправляет пунктуацию и улучшает формулировки. Готово к использованию немедленно.
- **Глобальные горячие клавиши** — Работает в любом приложении, поддерживает режимы удержания (Hold) и переключения (Toggle).
- **Низкая задержка** — Работает на базе движка логического вывода Groq, время обработки от начала до конца менее 3 секунд (включая ИИ-обработку).
- **Словарь пользователя** — Обеспечивает точную транскрипцию имен собственных и технических терминов.
- **История и статистика** — Автоматически сохраняет все транскрипции, есть Dashboard (Панель управления) для обзора использования.
- **Минимальная настройка** — Просто настройте ключ API, чтобы начать.

### Требования

- [Groq API Key](https://console.groq.com/keys) (Бесплатная регистрация и получение заявки)

### Быстрый старт

1. Скачайте и установите.
2. Откройте TypeK → Настройки → Вставьте свой Groq API Key.
3. В любом приложении зажмите горячую клавишу, чтобы начать говорить, отпустите, и текст автоматически вставится.

## Техническая архитектура

```
Tauri v2 (Rust) + Vue 3 + TypeScript

  ┌──────────────────────────────────┐
  │        Tauri Backend (Rust)      │
  │ Гор. клавиши · Буфер обмена      │
  └───────┬──────────────┬───────────┘
          │ invoke()     │ emit()
  ┌───────▼──┐    ┌──────▼───────────┐
  │   HUD    │    │    Dashboard     │
  │ Статус   │    │ История/Стат-ка  │
  └──────────┘    └──────────────────┘
```

- **Frontend** — Vue 3 + TypeScript + shadcn-vue + Tailwind CSS
- **Backend** — Rust (Tauri v2)
- **AI** — Groq Whisper (Голос в текст) + Groq LLM (Улучшение текста)
- **Хранилище** — SQLite (История) + tauri-plugin-store (Настройки)
