# TypeK - أداة إدخال الصوت

[اقرأ بلغات أخرى: [繁體中文](README.md) | [English](README_en-US.md) | [日本語](README_ja-JP.md) | [简体中文](README_zh-CN.md) | [한국어](README_ko-KR.md) | [Français](README_fr-FR.md) | [Español](README_es-ES.md) | [Русский](README_ru-RU.md) | [ไทย](README_th-TH.md) | [Tiếng Việt](README_vi-VN.md) | [العربية](README_ar-SA.md)]

> اضغط مع الاستمرار للتحدث، ثم أفلت لتجربة جودة مخرجات مختلفة تمامًا — أداة سطح مكتب لتحويل الصوت إلى نص.

TypeK هي أداة إدخال صوتي على أجهزة الكمبيوتر تعمل عبر منصات متعددة. اضغط مع الاستمرار على مفتاح الاختصار للتحدث في أي تطبيق، ثم أفلت ليتم نسخ الصوت عبر واجهة برمجة تطبيقات Groq Whisper، وبعدها يقوم Groq LLM تلقائيًا بتحويل كلامك المسموع إلى نص مكتوب وطلاقة ولصقه مباشرة في موضع المؤشر — وكأن لديك سكرتير تحرير مخفي يقوم بتعديل كلامك إلى كتابة رسمية.

## الميزات

- **من الكلام المنطوق إلى المكتوب** — يزيل الذكاء الاصطناعي الكلمات الزائدة، ويعيد تنظيم بنية الجملة، ويصلح علامات الترقيم، وينقح الصياغة. جاهز للاستخدام الفوري.
- **مفاتيح اختصار عامة** — يعمل في أي تطبيق، ويدعم كلا من وضعي الاستمرار (Hold) والتبديل (Toggle).
- **سرعة استجابة عالية** — مدعوم بمحرك الاستنتاج Groq، ومن البداية للنهاية في أقل من 3 ثوانٍ (بما في ذلك معالجة الذكاء الاصطناعي).
- **قاموس مفردات مخصص** — يضمن النسخ الدقيق للأسماء العلم والمصطلحات التقنية.
- **التاريخ والإحصاءات** — يحفظ تلقائيًا كل النصوص المكتوبة، مع لوحة معلومات لعرض الاستخدام بلمحة سريعة.
- **إعداد بسيط للغاية** — فقط قم بتكوين مفتاح واجهة برمجة التطبيقات (API Key) للبدء.

## التثبيت

### التنزيل

| النظام | رابط التنزيل |
|------|---------|
| macOS (Apple Silicon) | [TypeK_0.9.1_aarch64.dmg](https://github.com/kekukeku/TypeK/releases/download/v0.9.1/TypeK_0.9.1_aarch64.dmg) |
| Windows (x64) | [TypeK_0.9.1_x64.exe](https://github.com/kekukeku/TypeK/releases/download/v0.9.1/TypeK_0.9.1_x64.exe) |

> **⚠️ ملاحظة حول التثبيت على نظام macOS (خطأ "التطبيق تالف"):**
> إذا رأيت خطأ "تالف" عند فتح التطبيق على Mac، يرجى تنزيل وتشغيل البرنامج النصي [Unlock_TypeK.command](https://raw.githubusercontent.com/kekukeku/TypeK/main/%E8%A7%A3%E9%8E%96TypeK.command) لإزالة قيود العزل تلقائيًا، أو افتح الجهاز الطرفي (Terminal) وقم بتشغيل: `sudo xattr -cr /Applications/TypeK.app`.

### المتطلبات

- [مفتاح Groq API](https://console.groq.com/keys) (سجل للحصول عليه مجانًا)

### البدء السريع

1. قم بالتنزيل والتثبيت.
2. افتح TypeK → صفحة الإعدادات → الصق مفتاح Groq API الخاص بك.
3. اضغط مع الاستمرار على مفتاح الاختصار في أي تطبيق للتحدث، ثم أفلت حتى يتم لصق النص تلقائياً.

## الهيكلية التقنية

```
Tauri v2 (Rust) + Vue 3 + TypeScript

  ┌──────────────────────────────────┐
  │        Tauri Backend (Rust)      │
  │ اختصارات عامة · الحافظة · الصوت  │
  └───────┬──────────────┬───────────┘
          │ invoke()     │ emit()
  ┌───────▼──┐    ┌──────▼───────────┐
  │   HUD    │    │    Dashboard     │
  │  الحالة  │    │ الإعدادات/التاريخ│
  └──────────┘    └──────────────────┘
```

- **واجهة المستخدم** — Vue 3 + TypeScript + shadcn-vue + Tailwind CSS
- **الخلفية** — Rust (Tauri v2)
- **الذكاء الاصطناعي** — Groq Whisper (من الصوت إلى النص) + Groq LLM (تحسين النص)
- **التخزين** — SQLite (التاريخ) + tauri-plugin-store (الإعدادات)

## التطوير

### متطلبات البيئة

- Node.js 24+
- pnpm 10+
- Rust stable
- Xcode Command Line Tools (macOS)

### الأوامر

```bash
# تثبيت التبعيات
pnpm install

# وضع التطوير
pnpm tauri dev

# البناء (Build)
pnpm tauri build

# الاختبار
pnpm test

# فحص نوع البيانات
npx vue-tsc --noEmit
```

### النشر المطوّر

```bash
./scripts/release.sh 0.2.0
# → تحديث تلقائي للنسخة والمساهمة وغيرها
# → GitHub Actions تقوم ببناء المثبتات لأجهزة macOS و Windows
# → اذهب إلى GitHub Releases لنشر يدوي
```
