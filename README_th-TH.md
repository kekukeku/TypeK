# TypeK - เครื่องมือป้อนข้อมูลด้วยเสียง

[อ่านในภาษาอื่น: [繁體中文](README.md) | [English](README_en-US.md) | [日本語](README_ja-JP.md) | [简体中文](README_zh-CN.md) | [한국어](README_ko-KR.md) | [Français](README_fr-FR.md) | [Español](README_es-ES.md) | [Русский](README_ru-RU.md) | [ไทย](README_th-TH.md) | [Tiếng Việt](README_vi-VN.md) | [العربية](README_ar-SA.md)]

> กดค้างไว้เพื่อพูด ปล่อยเพื่อสัมผัสคุณภาพเอาต์พุตที่แตกต่างอย่างสิ้นเชิง — เครื่องมือแปลงเสียงเป็นข้อความบนเดสก์ท็อป

TypeK เป็นเครื่องมือป้อนข้อมูลด้วยเสียงข้ามแพลตฟอร์ม กดแป้นพิมพ์ลัดค้างไว้เพื่อพูดในแอปพลิเคชันใดก็ได้ ปล่อยเพื่อถอดเสียงผ่าน Groq Whisper API จากนั้น Groq LLM จะแปลงภาษาพูดของคุณเป็นข้อความที่เขียนอย่างสละสลวยโดยอัตโนมัติ และวางไว้ที่ตำแหน่งเคอร์เซอร์โดยตรง—ราวกับมีเลขานุการแก้ไขที่มองไม่เห็นมาแก้ไขคำพูดให้เป็นภาษาทางการ

## คุณสมบัติ

- **จากเสียงพูดเป็นตัวอักษร** — AI ลบคำสร้อยหรือคำที่ไม่มีความหมาย ปรับรูปประโยคใหม่ แก้ไขเครื่องหมายวรรคตอน และเกลาคำพูดโดยอัตโนมัติ พร้อมใช้งานทันที
- **คีย์ลัดครอบคลุม** — ใช้งานในแอปพลิเคชันใดก็ได้ รองรับทั้งโหมดกดค้าง (Hold) และสลับหน้า (Toggle)
- **เวลาแฝงต่ำ** — ขับเคลื่อนด้วยเครื่องมือ Groq สิ้นสุดภายใน 3 วินาที (รวมการประมวลผลของ AI)
- **พจนานุกรมส่วนตัว** — ตรวจสอบการถอดเสียงชื่อเฉพาะและศัพท์เทคนิคให้ถูกต้อง
- **ประวัติและสถิติ** — บันทึกผลการถอดเสียงโดยอัตโนมัติ พร้อมแดชบอร์ดสำหรับดูผลการใช้งาน
- **การตั้งค่าที่ง่ายดาย** — เพียงตั้งค่าคีย์ API ก็เริ่มใช้งานได้

## การติดตั้ง

### ดาวน์โหลด

| แพลตฟอร์ม | ลิงก์ดาวน์โหลด |
|------|---------|
| macOS (Apple Silicon) | [TypeK_0.9.0_aarch64.dmg](https://github.com/kekukeku/TypeK/releases/download/v0.9.0/TypeK_0.9.0_aarch64.dmg) |
| Windows (x64) | [TypeK_0.9.0_x64.exe](https://github.com/kekukeku/TypeK/releases/download/v0.9.0/TypeK_0.9.0_x64.exe) |

> **⚠️ หมายเหตุการติดตั้งสำหรับ macOS (ข้อผิดพลาด "แอปพลิเคชันเสียหาย"):**
> หากคุณพบข้อผิดพลาด "เสียหาย" เมื่อเปิดแอปบน Mac โปรดดาวน์โหลดและเรียกใช้สคริปต์ [Unlock_TypeK.command](https://raw.githubusercontent.com/kekukeku/TypeK/main/%E8%A7%A3%E9%8E%96TypeK.command) เพื่อลบข้อจำกัดการกักกันออกโดยอัตโนมัติ หรือเปิด Terminal แล้วรัน: `sudo xattr -cr /Applications/TypeK.app`.

### ความต้องการของระบบ

- [Groq API Key](https://console.groq.com/keys) (ลงทะเบียนขอได้ฟรี)

### เริ่มต้นง่ายๆ

1. ดาวน์โหลดและติดตั้ง
2. เปิด TypeK → ไปที่ตั้งค่า → กรอก Groq API Key
3. กดคีย์ลัดค้างไว้ในแอปใดก็ได้ แล้วพูด ปล่อยเพื่อวางแบบอัตโนมัติ

## โครงสร้างทางเทคนิค

```
Tauri v2 (Rust) + Vue 3 + TypeScript

  ┌──────────────────────────────────┐
  │        Tauri Backend (Rust)      │
  │ ฮอตคีย์ทั่วโลก · คลิปบอร์ด · เสียง  │
  └───────┬──────────────┬───────────┘
          │ invoke()     │ emit()
  ┌───────▼──┐    ┌──────▼───────────┐
  │   HUD    │    │    Dashboard     │
  │ สถานะ    │    │ ประวัติ/สถิติ       │
  └──────────┘    └──────────────────┘
```

- **Frontend** — Vue 3 + TypeScript + shadcn-vue + Tailwind CSS
- **Backend** — Rust (Tauri v2)
- **AI** — Groq Whisper (เสียงเป็นข้อความ) + Groq LLM (ปรับข้อความ)
- **Storage** — SQLite (ประวัติ) + tauri-plugin-store (ตั้งค่า)

## การพัฒนาโปรแกรม

### สิ่งแวดล้อมระบบ

- Node.js 24+
- pnpm 10+
- Rust stable
- Xcode Command Line Tools (macOS)

### คำสั่ง

```bash
# ติดตั้งแพ็กเกจ
pnpm install

# โหมดออกแบบ (Dev)
pnpm tauri dev

# สร้าง (Build)
pnpm tauri build

# ทดสอบ
pnpm test

# กรองรูปแบบข้อมูล
npx vue-tsc --noEmit
```

### การเผยแพร่

```bash
./scripts/release.sh 0.2.0
# → อัปเดตเวอร์ชัน คอมมิต และ Push 
# → GitHub Actions สร้างตัวติดตั้งบน macOS + Windows
# → ปล่อย Publish ด้วยตัวเองที่ GitHub Releases
```
